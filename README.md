## Introduction:

Web App feitá para o teste para a vaga de Front-End na CotaBox

## Ferramentas:
  Vue3	<br>
  Sass <br>
  Vue-chartjs V2
 
## Link Para ver a aplicação rodando na Netlify

https://cotaboxtest.netlify.app

## Start:

```sh
$ git clone https://github.com/brunofariasdev/CotaBoxTest.git
```

```sh
$ cd CotaBoxTest
```

```sh
$ npm install
```

```
$ npm run serve
```
